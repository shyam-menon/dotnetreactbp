# Common Commands

## Backend (.NET)

```bash
# Restore dependencies
dotnet restore

# Build the project
dotnet build

# Run the project
dotnet run

# Create a new controller
dotnet new controller -n YourControllerName -o Controllers

# Run tests
dotnet test
```

## Frontend (React)

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Add a package
npm install package-name
```

## Git Commands

```bash
# Check status
git status

# Create a new branch
git checkout -b feature/new-feature

# Switch branches
git checkout branch-name

# Commit changes
git add .
git commit -m "Your commit message"

# Push changes
git push

# Pull latest changes
git pull
```
